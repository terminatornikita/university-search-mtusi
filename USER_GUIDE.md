# Руководство пользователя

## 1. Загрузка документов

Нажмите кнопку «Выбрать файлы» и выберите PDF или DOCX. Дождитесь статуса «Готово».

## 2. Поиск

Введите запрос в поле поиска и нажмите «Найти» или Enter. Результаты отобразятся в виде карточек с подсветкой совпадений.

## 3. Просмотр результатов

Каждая карточка содержит название файла, номер страницы, фрагмент текста и оценку релевантности.
